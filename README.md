# Contact Binary Workbench（CBW）
CBW is a tool designed for the rapid orbital solution of binary systems, enabling the joint analysis of radial velocity and light curves. We provide two executable (EXE) versions, in which the embedded radial velocity models are constructed using a fully connected neural network and a SIREN architecture, respectively. Users may select the appropriate version according to their specific requirements.
The directory is used to store the light-curve and radial velocity data, with the data format following that described in the paper A Neural-network Model for Quickly Solving Multiple-band Light Curves of Contact Binaries. The radial velocity file should be named rv.csv, and an example is provided for reference. The CSV and JSON files are used for configuring the parameter search ranges and control switches, respectively, which can also be set in accordance with the aforementioned paper.
